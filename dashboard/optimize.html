Indicator lengths — editable choice lists
Indicator lengths can't be searched as a continuous range (each length means recomputing the indicator over all history), so the search picks from these lists instead. The lists themselves are yours to edit — add or change any lengths below, hit Save. The first run after a change rebuilds the engine cache once (V5 family: a few minutes; Scalp family: ~10s). Checkboxes narrow which options the search may pick without rebuilding.

Guide

Click any label ⓘ, column header, or action button name to see what it means and how to use it.

Quick start: keep the defaults (V7 · genetic · lev · vol3 · per-regime, train-end 2025-09-01), set Processes to your CPU cores, pick hours, hit Start search. When it finishes, look at the holdout column — green means it made money on data it never saw. Then Backtest it (preview appears here) and Adopt it into the trader.
